How to play this game.

Make a word out of falling blocks atleast 3 letter Word.

Mobile User

  • Swipe left to move left.
  • Swipe right to move right.
  • Swipe down to move down.
  • Swipe up to make the sit to the bottom.
  • Tap the falling block to hold.
  • Tap the block on the hold section to release it and swap with the current block.
Desktop User
  • Arrow left or 'a' move left.
  • Arrow right or 'd' to move right.
  • Arrow down or 's' to move down.
  • Arrow up or 'w' to make the sit to the bottom.
  • Double tap to hold.
  • Press 1,2,3 to release hold block item and swap with the current block. Note that the first block on the left is one and so on.
